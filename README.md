# Password Generator
A Python-based password generator program that creates secure passwords using letters, numbers and symbols with customizable length options.
## Features
- Generates random passwords with customizable length
- Includes uppercase and lowercase letters
- Includes numbers and special symbols
- Option to shuffle characters for better security

## How to use
1. Make sure you have Python already installed
2. Download this repository
3. Run the script
4. Follow the prompts to customize your password

## Sample output

How many letters do you want in your password?

How many numbers do you want in your password?
